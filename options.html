How It Works

  1. Add the base URL of each GitLab instance you use.
  2. Chrome will ask for permission to run on that host — accept it.
  3. Visit any project's /-/tags page on that instance.
  4. An icon appears next to each tag — click it to generate a changelog.

The extension uses your existing logged-in session (cookies). No API tokens needed.
